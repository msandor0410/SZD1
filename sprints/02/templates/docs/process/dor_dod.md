# Definition of Ready (DoR) és Definition of Done (DoD)

## Definition of Ready (DoR)
Egy feladat akkor "Ready", ha:
- Megvan a hozzá tartozó User Story
- Megvannak az AC-k (Given–When–Then)
- A függőségek azonosítva vannak
- A szükséges UI flow körvonala (wireframe) megvan
- A feladat becsülhető és önállóan végrehajtható

## Definition of Done (DoD)
Egy feladat akkor "Done", ha:
- Minden AC teljesül
- A kód átment a lokális builden
- Van legalább 1 unit teszt (ha releváns)
- A coverage nem csökken (Sprint 2 minimum: 60%)
- A CI minden lépése zöld
- A funkció része a smoke ellenőrzésnek
- Dokumentáció frissítve (Spec, Story, Traceability)
